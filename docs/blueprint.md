# **App Name**: AI Hub

## Core Features:

- AI Tool Discovery & Search: Browse and search a categorized catalog of AI tools with essential details and basic filtering options.
- Secure User Authentication: Register and log in via Email/Password and Google Sign-In, with OTP email verification for secure account actions.
- Tool Ratings & Reviews: Users can submit text reviews and star ratings for AI tools, providing community feedback.
- Localized User Interface: Support for English, Arabic (with full RTL support), and French languages for a global user experience.
- AI-Powered Tool Recommendation: A generative AI tool provides personalized AI tool recommendations based on user viewing history or expressed interests.
- New Tool Submission Form: Registered users can submit new AI tools for administrative review and approval.
- Admin Tool & User Management: A basic admin interface to review and manage submitted AI tools and user accounts.

## Style Guidelines:

- Overall color scheme: Dark. Chosen for a sophisticated, modern, and tech-focused aesthetic suitable for a global AI platform.
- Primary color: A deep, professional bluish-teal (#2E93B8). This hue represents intelligence and innovation, standing out elegantly against a dark background.
- Background color: A very dark grey-blue (#20282B). This desaturated dark base provides excellent contrast and emphasizes the platform's professional nature.
- Accent color: A vibrant aqua-green (#4ACED7). This analogous color provides a refreshing highlight, drawing attention to interactive elements and calls to action.
- Headline and body font: 'Inter' (sans-serif). Chosen for its neutrality, excellent legibility, and modern appearance, ensuring clarity across various text types and languages.
- Use clean, modern, and highly recognizable SVG icons to ensure crisp display across all devices and to convey information clearly.
- Responsive and adaptive layout designed to provide an optimal viewing and interaction experience across desktops, tablets, and mobile devices, including full RTL support for Arabic.
- Incorporate subtle and smooth transitions for page loading, navigation, and interactive elements to enhance user experience without distraction.
